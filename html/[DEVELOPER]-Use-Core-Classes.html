Use Core Classes

Send a Response

Use the Phile\Core\Response class to send an answer to the browser. You can easily send your own response:

// sends 'Hello World' to the browser
$response = new \Phile\Core\Response;
$response->setBody('Hello World')->send();

To reuse ore manipulate Phile's core response listen to the after_init_core event.

Access the Session

Phile\Core\Session allows easy access to the Session environment. For example

// set a session variable
Session::set('isLoggedIn', true);

// read a session variable
$lastVisitedPage = Session::get('lastVisitedPage');
